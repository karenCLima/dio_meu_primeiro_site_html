Fundamentos do HTML

Voltar

Todo HTML tem uma estrutura básica para que você possa digitar as tags. Abaixo temos um exemplo de um documento padrão:

		<!DOCTYPE html>
		<html>
		<head>
		    <title>Título da Página</title>
		    <meta charset = "utf-8"/>
		</head>
		<body>

		</body>
		</html>

Entre essas tags podemos destacar a tag <title></title>; que define o nome da pagina que aparece na aba do navegador e a tag <body></body> onde fica localizado o corpo do documento, ou seja, tudo o que vai ser vizualizado por quem navega na sua pagina.

O que são Tags?

Voltar

As tags são os elementos usados dentro do HTML pra informar o navegador a estrutura e o conteúdo visual do site. Sem elas, o navegador não teria como imterpretar.

As tags sempre vêm dentro de sinais conhecidos como chevron (<>) para iniciar uma tag e (</>) para fechar uma tag. As tags podem ser divididas entre tags que precisam de fechamento e tags que não precisam de fechamento.Podem existir também, tags dentro de tags: <tag1><tag2>elemento</tag2></tag1>

Além disso, as tags podem possuir atributos que modificam a sua vizualição ou modificam a sua forma de funcionamento.

O que são Atributos?

Voltar

Atributos são elementos utilizados para personalizar as tags. Eles também são utilizados para diferenciar tags para aplicação de diferentes formatações.

Exemplos de Atributos mais utilizados:

  • id ="": Atribui um identificador único ao elemento.
  • class ="": Atribui uma classe ao elemento.
  • style ="": Permite alterar o estilo do elemento. Também permite a inclusão de elementos CSS.
  • Width ="": Define um tamanho para o elemento.

Tags de Texto

Voltar

Existem diversas tags para definir textos. Vamos ver as principais tags a seguir:

  • Tag de título <h></h>

    São as tags reponsáveis pelos títulos, subtítulos e títulos das seções da pagina. Elas vão de h1 a h6 e têm valor hieráquico, sendo o h1 com maior importância e h6 com menor importância. 

    				<h1>Título1</h1>
				<h2>Subtítulo1</h2>
				<h3>Subtítulo2</h3>
				<h4>Subtítulo3</h4>
				<h5>Subtítulo4</h5>
				<h6>Subtítulo5</h6>

  • Tag <p></p>: É a tag que define paragrafos. 

    				<p>Aqui vai um texto.</p>
				Vizualização:Aqui vai um texto.

  • Tag <hr/>: Serve para formar uma linha horizontal. Ela não precisa de fechamento. 

    				<p>Aqui vai um texto</p>
				<hr/>
				<p>Aqui vai outro texto</p>
				
    Vizualização:
    
				
    Aqui vai um texto.
    
				
     
				
    Aqui vai outro texto.

  • Tag <mark></mark>: Faz uma formatação do tipo marca-texto na linha. 

    				<mark>Aqui vai um texto marcado.</mark>
				 Vizualização:Aqui vai um texto marcado.

  • Tag <i></i>: Faz uma formatação do tipo itálico no texto. 

    				<i>Aqui vai um texto em itálico.</i>
				 Vizualização:Aqui vai um texto em itálico.

  • Tag <u></u>: Faz uma formatação do tipo sublinhado no texto. 

    				<u>Aqui vai um texto sublinhado.</u>
				 Vizualização:Aqui vai um texto sublinhado.

  • Tag <strong></strong>: Faz uma formatação do tipo negrito no texto. 

    				<strong>Aqui vai um texto em negrito.</strong>
				 Vizualização:Aqui vai um texto em negrito.

  • Tag <small></small>: Faz uma formatação do tipo diminuição de fonte no texto. 

    				<small>Aqui vai um texto com fonte reduzida.</small>
				 Vizualização:Aqui vai um texto com fonte reduzida.

  • Tag <sub></sub>: Faz uma formatação do tipo subscrito no texto. 

    				<p>Aqui vai um texto <sub>com subscrito.</sub></p>
				 Vizualização:Aqui vai um texto com subscrito.
				<p>a<sub>2</sub></p>
				 Vizualização: a 2

  • Tag <sup></sup>: Faz uma formatação do tipo sobrescrito no texto. 

    				<p>Aqui vai um texto <sup>com sobrescrito.</sup></p>
				 Vizualização:Aqui vai um texto com sobrescrito.
				<p>a<sup>2</sup></p>
				 Vizualização: a 2

  • Tag <blockquote></blockquote>: Faz uma formatação do tipo citação. 

    				<blockquote>Aqui vai um texto em forma de citação.</blockquote>
				 Vizualização:
    Aqui vai um texto em forma de citação

  • Tag <font></font>: Faz uma formatação no tamanho, na fonte e na cor do texto. Atualmente, foi substituído pelo uso de propriedades CSS. 

    				<font color="red" face="Vedana, Geneva, sans-serif" size="+1">Aqui vai um texto formatado.</font>
				 Vizualização:Aqui vai um texto formatado.

  • Tag <del></del>: Mostra uma parte deletada do documento. Ele pode ou não aparecer renderizada no navegador como um texto riscado. 

    				<p><del>Aqui vai um texto deletado.</del></p>
				 Vizualização:Aqui vai um texto deletado.

  • Tag <abbr></abbr>: Representa uma abreviação de alguma palavra. Você pode descrever a palavra abreviada no atributo title. 

    				<p>Biden é presidente dos <abbr title="Estados Unidos da America">EUA.</abbr></p>
				 Vizualização: Biden é presidente dos EUA.

Tags para listas

Voltar

Para criar listas dentro do HTML temos que usar tags específicas para listas. Existem dois tipos de listas: listas ordenadas e listas não ordenadas. Vamos ver cada uma delas a seguir:

  • Listas Ordenadas: são listas que seguem uma ordem numérica. Usamos <ol></ol> para criar uma lista ordenada e as tags <li></li> para criar elementos dentro da lista. 

    
				<ol> 
				<li>Item 1</li>
				<li>Item 2</li>
				<li>Item 3</li>
				</ol>
    Vizualização:
    1. Item 1
    2. Item 2
    3. Item 3

  • Listas Não-Ordenadas: são listas que não seguem uma ordem numérica. Usamos <ul></ul> para criar uma lista não-ordenada e as tags <li></li> para criar elementos dentro da lista. 

    
				<ul> 
				<li>Item 1</li>
				<li>Item 2</li>
				<li>Item 3</li>
				</ul>
    Vizualização:
    • Item 1
    • Item 2
    • Item 3

Tag para Link

Voltar

Essa tag insere um hiperlink para outra página ou para outra página do mesmo domínio. Ele é representado pela tag <a href="insira aqui a URL"></a>. Ao colocar o atributo target como "_blanc" ele abre a página em outra aba do navegado quando é acessado, se for "_self" abre na mesma aba.

Você também pode linkar um subtítulo de uma página. Para isso, basta você colocar no href o id do subtítulo, desta forma: href= "#id_do_Subtítulo".

		<a href= "https://www.dio.me" target="_blanc">Site da DIO.me</a>
		Vizualização:Site da DIO.me

Tag para Imagem

Voltar

É utilizada para inserir uma imagem na página.Ele é representado pela tag <img src="insira aqui a URL da imagem"/>. Essa tag não possue fechamento. O atributo alt define um texto alternativo para a imagem. E os atributos height e width definem o tamanho. 

		<img src= "https://www.imagensempng.com.br/wp-content/uploads/2020/12/Logo-Google-Png-1024x1024.png" alt="Logo do Google" heiht="200px" width="200px">
